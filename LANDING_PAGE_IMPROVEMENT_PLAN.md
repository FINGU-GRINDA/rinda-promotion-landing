## 📊 패스트캠퍼스 페이지 핵심 성공 요소

### 1. **설득 흐름 (Conversion Funnel)**
```
감정적 공감 → 문제 인식 → 구체적 솔루션 → 시각적 증거 → 세부 커리큘럼 → 신뢰도 구축 → 긴급성 조성 → 행동 유도
```

### 2. **비주얼 전략**
- **GIF 애니메이션**: 도구 사용 방법 시연 (실제 화면 캡처)
- **3-4열 캐러셀**: 정보 밀도 높이면서 공간 효율화
- **스크린샷**: ChatGPT, Canva 등 구체적 도구 사용 화면
- **결과물 미리보기**: 완성된 전자책 표지, 페이지 예시

### 3. **카피라이팅 패턴**
| 패턴 | 예시 | 효과 |
|------|------|------|
| 감정적 자극 | "AI로 쓰는 내 인생 역전 시나리오" | 감정적 연결 |
| 구체적 약속 | "하루 30분, 30일이면 완성" | 명확한 기대치 설정 |
| 불안감 해소 | "챗봇이 도와드려요" | 진입 장벽 낮춤 |
| 수량/시간 제한 | "단 10좌석", "스터디 마감 임박" | 긴급성 |
| 사회적 증거 | 구매자 전용 혜택 | 신뢰도 |

### 4. **색상 심리**
- **검정/흰색**: 신뢰감, 전문성, 고급스러움
- **주황색 (#F54F35)**: CTA 버튼, 긴급성, 행동 유도
- **베이지/옅은 핑크**: 친근함, 접근성

## 🚀 개선 필요 영역

### ❌ 부족한 부분

1. **시각적 증거 부족**
   - 프로그램 진행 화면 없음
   - 성공 사례의 구체적 비주얼 부족
   - 도구 사용 예시 없음

2. **사회적 증거 약함**
   - 고객 후기 1개만 존재 (Hero에만)
   - 기업 로고, 숫자 통계 없음
   - Before/After 비교 부족

3. **정보 밀도 낮음**
   - 한 화면에 정보량이 적어 스크롤이 과도하게 길어짐
   - 공백이 많아 모바일에서 집중력 저하

4. **단계별 프로세스 시각화 부족**
   - Solution 섹션이 텍스트 위주
   - "어떻게 진행되는지" 이미지로 보여주지 않음
